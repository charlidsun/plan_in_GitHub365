# GitHub365计划

此举主要目的是为了复习以前的知识，并对自己的自律进行一项挑战。

所谓的365计划就是从 2015年1月25起,
每天在GitHub发布一篇正确的代码。一共持续365天。
每天都会记录在此笔记本上。

文件是在Mac OS X 终端下的 VIM 进行书写，测试完毕后将源代码上传。

补充,用Eclipes 编写。
